# QxBin Logic Implementations

Rupesh Malpani's QxBin framework: Room-temperature personal cubit simulation using Binary Probability Matrices, fractional states, exponents for superposition-like behavior.

## Two Tiers

1. **qxbin_edge.py** - Edge/Desktop: Personal simulator for education, prototyping.
2. **qxbin_cloud.py** - Server/Cloud: Scalable batch processing with parallelism.

## Run

```bash
pip install numpy matplotlib numba
python qxbin_edge.py
python qxbin_cloud.py
```

Inspired by coin-toss probabilities, grids, punch cards. Democratize quantum-like logic!
